# Contributing to MK Portfolio

Thank you for your interest in contributing! Here's how to get started.

---

## Getting Started

```bash
git clone https://github.com/Mohammedkarim-MK/mk-portfolio.git
cd mk-portfolio
npm install
npm run dev
```

---

## Before You Submit

- Run `npm test` — all 73 tests must pass
- Run `npm run lint` — no new ESLint errors
- Run `npm run format` — code must be formatted with Prettier

---

## Guidelines

- Keep pull requests focused — one change per PR
- Write clear commit messages: `fix: correct image path in about.html`
- Never commit API keys or secrets — use environment variables
- Test your changes locally before submitting

---

## Commit Message Format

```
type: short description

feat:     new feature
fix:      bug fix
docs:     documentation only
style:    formatting, no logic change
refactor: code restructure, no behaviour change
test:     adding or updating tests
ci:       CI/CD pipeline changes
```
